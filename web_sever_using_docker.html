Setting up a Web Server with Docker

Overview

The goal of this page is to provide a basic tutorial of how to set up a web server on your local computer using Docker.

Tutorial

  1. Install Docker
  2. After installing Docker, run a test to see that the installation was a success. To do so, type “docker run hello-world” and you should see "Hello from Docker!", among other helpful tips
  3. Next, pull an Ubuntu image from Docker using “docker pull ubuntu”. You should see that it has "Download newer image for ubuntu:latest", among other lines.
  4. To see that the Ubuntu image was pulled correctly, type “docker images” to check the existing Docker images on your local computer. For example, you should see the “ubuntu” image, which indicates that it was successfully pulled.
  5. Now, we are going to run the Ubuntu image to start a Docker container, and then log into this Ubuntu container. Type “docker run -it ubuntu:latest /bin/bash”, and check to see if a bash command prompt pops up. This command first runs docker, creates an interactive shell with “-it”, and uses the ubuntu docker image (:latest indicates the TAG). In this case, the string after "root@" is the container ID.
  6. Install the necessary applications into this Ubuntu container with “apt-get update”. If this fails, restart Docker.
  7. Next, install apache2 with “apt-get install apache2”.
  8. To edit the html files, install vim editor with “apt-get install vim”
  9. To exit the Ubuntu container, type “exit”
  10. To check the container, use “docker ps -a”
  11. To commit the container and create a new Docker image, type ”docker commit ubuntu-my_apache2”. The is the same as discussed before, and “ubuntu-my_apache2” is the name of the new Docker image.
  12. To check the new images, use “docker images”
  13. To share data between the Docker container and the host, type “docker run -it -p 127.0.0.1:80:80 -v C:\DockerApache:/a_dir_in_container ubuntu-my_apache2 /bin/bash”. “-p” exposes a port of the Docker container, and “-v” binds a host directory to the Docker container.
  14. Use the command "pwd" to check the present working directory, and then go into the html directory "/var/www/htm" with "cd /var/www/htm"
  15. Use "vi index.html" to create the "index.html" page. To insert into the file, use the "i" key (it will show --INSERT--)
  16. Once you have inserted your code, hit the escape key and then type ":wq" to write to the file and quit
  17. Now that the index.html page has been created, use "/etc/init.d/apache2 restart" in your Docker container to start the apache service
  18. Within the browser of your choice, access "127.0.0.1" and you can see your website.
