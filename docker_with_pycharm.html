Docker with Pycharm

Overview

This page aims to help users in getting comfortable with using Docker within Pycharm.

Tutorial

  1. Make sure PyCharm and Git are downloaded, and ensure Docker is running.
  2. Creat a new directory where your python script will reside
  3. Within this new directory, create a new python script with any program you want
  4. Create a new file "Dockerfile". It is important that there is no file extension with the Dockerfile
  5. Within this Dockerfile, write "FROM python:3", "ADD [dir_name] /[dir_name]", "RUN pip install *", "CMD ["python", "/src/[python_name.py]"]". Each new command is on a different line.
    • "FROM python:3": install Python 3
    • "ADD [dir_name] /[dir_name]": take the code in [dir_name], and move it to a [dir_name] folder
    • "RUN pip install *": depending on your program, this is where you install all of your needed packages
    • "CMD ["python", "/src/[python_name.py]"]": run the python file from the given directory
  6. To get PyCharm connected to the Docker server, go into "File"-->"Settings"-->"Build, Execution, Deployment"-->"Docker"-->"+". Once that is done, hit "Apply" and "Ok"
  7. Now, at the top click "Add Configuration"-->"+"-->"Docker"-->"Dockerfile". From here, change the name to whatever is relevant to your program, put in a value for "Image tag", and then hit "Apply"
  8. Hit the play button (Green play button at the top) to run the template
  9. Within this Docker console is the expected output of the program, along with other Docker items
  10. To add a new interpreter, go to the bottom of the page and find where it displays what version of python you are using "ex: Python 3.8", and then click on it and click "Add Interpreter"
  11. From here, go to "Docker"-->"OK", and then go back to the interpreter page and click on the new Docker interpreter
