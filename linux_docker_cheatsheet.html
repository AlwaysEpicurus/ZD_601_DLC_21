Linux & Docker Cheatsheet

Overview

This page aims to provide a user with some basic Linux commands.

Basic Commands

  1. pwd: find the path of the current working directory
  2. cd: change directory
    • Start with path "/home/user/Documents"
    • To access the subdirectory "Photos", use "cd Photos"
  3. ls: view contents of a directory
    • "ls /home/user/Documents" to view the contents of the "Documents" directory
  4. cat: list the contents of a file on the standard output
    • "cat example.txt" would output the contents of the specified file.
    • "cat > example.txt" would create a new file "example.txt"
  5. cp: copy files from currenty dir to a different dir
    • "cp ex.txt /home/user/Documents" would copy "ex.txt" to "/home/user/Documents"
  6. mv: move file, or rename file
    • "mv ex.txt /home/user/Documents" would move "ex.txt" to specified path
  7. mkdir: make a new directory
    • "mkdir example" would create new directory "example"
  8. rmdir: delete only an empty directory
    • "rmdir example" would remove this directory, only if it was empty
  9. rm: delete directory and content within it
  10. touch: create blank new file
    • "touch hello-world.txt" would create an empty file named "hello-world.txt"
  11. chmod: change the read, write, and execute permissions of files and directories
  12. chown: change or transfer ownership of a file to the specified user
    • "chown user1 ex.txt" would make user1 the owner of "ex.txt"

How to log into your Docker container

  1. Use format: "docker login [options] [server]
  2. Options: (--password, -p), (--username, -u)
  3. Examples:
    • docker login localhost:8080
    • docker exec -it [container_id] /bin/bash --> important to know the container id, list running containers to do so
